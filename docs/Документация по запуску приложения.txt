Инструкция по запуску приложения: 
	1)Открыть cmd или powershell если на Windows
	2)Вписать ssh root@адрес ( пример адреса -  102.321.22.123) и нажать enter
	3)Ввести пароль и нажать enter (пароль не будет показываться)
	4) написать cd commands и нажать enter
	5) Написать ./start.sh и нажать enter
	6) Должно написать что Запущен сервер и Prisma ORM
	7) Все готово, проверяйте

 ----------------------------------------------------------
	База данных находится по адресу http://адрес:5555 (адрес тот что вводили при подключении по ssh в пункте "Инструкция по запуску приложения") Обязательно http а не https

Инструкция по открытию доступа к базе данных:
	1)Открыть cmd или powershell если на Windows
	2)Вписать ssh root@адрес ( пример адреса -  102.321.22.123) и нажать enter
	3)Ввести пароль и нажать enter (пароль не будет показываться)
	4) написать cd commands и нажать enter
	5) Написать ./openPortDataBase.sh и нажать enter
	6) Все база данных открыта и вы можете в нее перейти, но не забудьте закрыть ее, как закрыть ее описано в "Инструкция по закрытию доступа к базе данных", инструкция приведена ниже

Инструкция по закрытию доступа к базе данных:
		1)Открыть cmd или powershell если на Windows
	2)Вписать ssh root@адрес ( пример адреса -  102.321.22.123) и нажать enter
	3)Ввести пароль и нажать enter (пароль не будет показываться)
	4) написать cd commands и нажать enter
	5) Написать ./closePortDataBase.sh и нажать enter
	6) Всё, доступ к базе данных закрыт, обязательно проверьте. Попробуйте зайти в базу, как указано в других инструкциях, если страница не доступна, не открывается, значит база данных закрыта, и к ней никому не удастся получить доступ
